# 💰 Smart Expense Tracker 

A modern Progressive Web Application (PWA) for tracking personal income and expenses with real-time synchronization, offline support, and intelligent expense entry using the Magic Bar.

## 🚀 Features

- 🔐 Secure User Authentication (Firebase)
- 💸 Income & Expense Tracking
- 📊 Real-Time Expense Visualization
- 📡 Real-Time Cloud Synchronization
- 📱 Progressive Web App (PWA)
- 🌐 Cross-Platform Support
- 📴 Offline-First Functionality
- 🧠 Magic Bar for Natural Language Expense Entry
- 🎤 Voice-Based Expense Input
- 📁 CSV Data Export
- 🌙 Light/Dark Theme Support
- ⚡ Fast & Responsive UI

---

# 🏗️ System Architecture

The application follows a three-tier architecture:

1. **Presentation Layer**
   - HTML5
   - CSS3
   - JavaScript

2. **Application Layer**
   - Service Workers
   - Firebase SDK
   - Business Logic

3. **Data Layer**
   - Cloud Firestore
   - IndexedDB
   - Firebase Authentication
